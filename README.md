## About Me

- Building Playwright/TypeScript automation suites and Maestro mobile flows for production releases.
- 4-tier POM enforcer — locators, pages, utils, tests. Zero raw selectors in spec files.
- 10-layer test methodology — UI, functional positive/negative, edge, network, lifecycle, permissions, input, push, E2E.
- BMAD operator — Build, Measure, Adjust, Deploy. Risk-weighted prioritization, not coverage theater.
- Reads diffs at speed. Treats edge cases like main characters.
